## Описание проекта
Консольное приложение на C#, имитирующее работу торгового автомата с функциями:
- Просмотр товаров с ценами и количеством
- Вставка монет разных номиналов (1₽, 2₽, 5₽, 10₽, 50₽, 100₽)
- Покупка товаров
- Выдача сдачи
- Режим администратора (пароль: admin123) для пополнения товаров и сбора денег

## Технологии
- C# .NET 8.0
- Объектно-ориентированное программирование
- Русский интерфейс

## Запуск проекта
```bash
dotnet run
```

-  Просмотр товаров (Кола - 150₽, Чипсы - 100₽, Шоколад - 80₽, Вода - 50₽)

-  Возврат монет
